[[Rogers Orders]]

[[Import Rogers Order via BUDDTL]]

Guidelines

Import a valid BUDDTL file, either as a CSV or plain text file format, where product AS400 IDs are provided along the top, Store IDs run down the first column, and the quantity of each product is shown in each row, such as: 

	999	998	997
123	0	1	1
124	1	0	100
Each store will generate a new order in the system.
